open class ListView<T>(height: Number, width: Number, posX: Number, posY: Number, items: List<T>, font: Font, orientation: Orientation, formatFunction: (T) -> String?) : UIElementView
Content copied to clipboard

A list view displaying its items next to each other in the given orientation. The formatFunction is used to gain a String representation of each item. If no formatFunction is specified the toString function gets used instead.

Parameters

height

height for this ListView. Default: 0.

width

width for this ListView. Default: 0.

posX

horizontal coordinate for this ListView. Default: 0.

posY

vertical coordinate for this ListView. Default: 0.

items

initial list of items for this ListView. Default: empty list.

font

font to be used for this ListView. Default: default Font constructor.

orientation

orientation for this ListView. Default: VERTICAL.

formatFunction

the formatFunction that is used to represent the items. Default: null.
