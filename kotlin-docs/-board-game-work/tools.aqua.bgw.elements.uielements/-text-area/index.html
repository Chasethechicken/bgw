TextArea

open class TextArea(height: Number, width: Number, posX: Number, posY: Number, text: String, font: Font, prompt: String) : LabeledUIElementView
Content copied to clipboard

Represents a TextArea that can be written to. Whenever user input occurs the textProperty field gets updated.

Parameters

height

Height for this textArea. Default: 0.

width

Width for this textArea. Default: 0.

posX

Horizontal coordinate for this TextArea. Default: 0.

posY

Vertical coordinate for this TextArea. Default: 0.

text

Initial text for this textArea. Default: empty string.

prompt

Prompt for this textArea. This gets displayed as a prompt to the user whenever the textProperty's value is an empty string. Default: empty string.
