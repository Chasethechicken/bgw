A color picker that allows to chose a color.

Parameters

height

height for this ColorPicker. Default: suggested color picker height COLOR_PICKER_HEIGHT.

width

width for this ColorPicker. Default: suggested color picker width COLOR_PICKER_WIDTH.

posX

horizontal coordinate for this ColorPicker. Default: 0.

posY

vertical coordinate for this ColorPicker. Default: 0.

initialColor

the color that is initially selected. Default: Color.WHITE.
