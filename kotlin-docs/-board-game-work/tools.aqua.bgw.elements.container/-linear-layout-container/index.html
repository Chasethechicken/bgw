LinearLayoutContainer

open class LinearLayoutContainer<T : GameElementView>(height: Number, width: Number, posX: Number, posY: Number, spacing: Number, visual: Visual, orientation: Orientation, alignment: Alignment) : GameElementContainerView<T>
Content copied to clipboard

A LinearLayoutContainer may be used to visualize a zone containing GameElementViews. GameElementViews inside the container get placed according to the specified Orientation and Alignment. A spacing between Elements may be specified which may also be negative e.g. Elements like playing cards should overlap.

Visualization: The Visual is used to visualize a background. If all Elements are still within bounds with the user defined spacing, the user defined spacing gets used to space the Elements. Otherwise the biggest possible spacing is used so that all Elements are still withing bounds of the LinearLayoutContainer.

Parameters

height

height for this LinearLayoutContainer. Default: 0.

width

width for this LinearLayoutContainer. Default: 0.

posX

horizontal coordinate for this LinearLayoutContainer. Default: 0.

posY

vertical coordinate for this LinearLayoutContainer. Default: 0.

spacing

spacing between contained GameElementViews. Default: 0.

orientation

orientation for this LinearLayoutContainer. Default: Orientation.HORIZONTAL.

alignment

specifies how the contained GameElementViews should be aligned. Default: Alignment.TOP_LEFT.
