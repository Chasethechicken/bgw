Content copied to clipboard

Defines an GameElementContainerView that orders elements in a grid structure.

Parameters

rows

initial row count.

columns

initial column count.

spacing

spacing between rows and columns. Default: 0.0

posX

horizontal coordinate for this GridLayoutView. Default: 0.

posY

vertical coordinate for this GridLayoutView. Default: 0.

layoutFromCenter

whether the GridLayoutView should anchor in the center (true) or top-Left (false). Default: true.

visual

initial visual for this GridLayoutView. Default: Visual.EMPTY.
