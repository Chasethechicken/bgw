DiceView

open class DiceView(height: Number, width: Number, posX: Number, posY: Number, visuals: MutableList<Visual>) : GameElementView
Content copied to clipboard

A DiceView may be used to visualize a dice.

Visualization: The Visual at the currentSide value is used to visualize the dice.

Parameters

height

height for this DiceView. Default: the suggested dice height.

width

width for this DiceView. Default: the suggested dice width.

posX

horizontal coordinate for this DiceView. Default: 0.

posY

vertical coordinate for this DiceView. Default: 0.

visuals

list of visuals to represent the sides of the die.
