ComponentView

abstract class ComponentView(posX: Number, posY: Number, width: Number, height: Number, visual: Visual)
Content copied to clipboard

ComponentView is the abstract baseclass of all framework components.

It defines important fields and functions that are necessary to visualize inheriting components.

See also

tools.aqua.bgw.exception.IllegalInheritanceException

Parameters

posX

the X coordinate for this ComponentView relative to its container.

posY

the Y coordinate for this ComponentView relative to its container.

width

width for this ComponentView.

height

height for this ComponentView.

visual

visual for this ComponentView.

Throws

tools.aqua.bgw.exception.IllegalInheritanceException

inheriting from this Class is not advised, because it cannot be rendered and trying to do so will result in an IllegalInheritanceException.
