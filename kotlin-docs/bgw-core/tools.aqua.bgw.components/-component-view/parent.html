parent

var parent: ComponentView? = null
Content copied to clipboard

The parent of this ComponentView. null if this ComponentView is not contained in a GameComponentContainer, LayoutView, or a Scene. If the component has been added directly to a Scene, parent is equal to the scene's RootComponent. If the component is contained within a container, parent is equal to that container.
