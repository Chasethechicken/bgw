GameComponentContainer

sealed class GameComponentContainer<T : GameComponentView> : DynamicComponentView, Iterable<T>

Baseclass for containers that can contain GameComponentViews or its subclasses.

It provides the list to store GameComponentViews and some useful methods to work on said list.
