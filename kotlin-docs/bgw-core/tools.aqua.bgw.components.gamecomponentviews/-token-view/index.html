open class TokenView(posX: Number, posY: Number, width: Number, height: Number, visual: Visual) : GameComponentView

A TokenView may be used to visualize any kind of token.

Visualization: The current Visual is used to visualize the token.

Parameters

posX

horizontal coordinate for this TokenView. Default: 0.

posY

vertical coordinate for this TokenView. Default: 0.

width

width for this TokenView. Default: DEFAULT_TOKEN_WIDTH.

height

height for this TokenView. Default: DEFAULT_TOKEN_HEIGHT.

visual

visual for this TokenView.
