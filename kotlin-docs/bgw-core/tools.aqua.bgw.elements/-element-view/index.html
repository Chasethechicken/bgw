ElementView

abstract class ElementView(height: Number, width: Number, posX: Number, posY: Number, visual: Visual) : Observable
Content copied to clipboard

ElementView is the abstract baseclass of all framework elements. It defines important fields and functions that are necessary to visualize inheriting elements.

See also

tools.aqua.bgw.exception.IllegalInheritanceException

Parameters

height

height for this ElementView.

width

width for this ElementView.

posX

the X coordinate for this ElementView relative to its container.

posY

the Y coordinate for this ElementView relative to its container.

visual

visual for this ElementView.

Throws

tools.aqua.bgw.exception.IllegalInheritanceException

inheriting from this Class is not advised, because it cannot be rendered and trying to do so will result in an IllegalInheritanceException.
