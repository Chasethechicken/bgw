open class ListView<T>(posX: Number, posY: Number, width: Number, height: Number, items: List<T>, font: Font, orientation: Orientation, formatFunction: (T) -> String?) : UIComponent
Content copied to clipboard

A ListView displaying its items next to each other in the given orientation.

The formatFunction is used to gain a String representation of each item. If no formatFunction is specified the toString function gets used instead.

Parameters

posX

Horizontal coordinate for this ListView. Default: 0.

posY

Vertical coordinate for this ListView. Default: 0.

width

Width for this ListView. Default: DEFAULT_LISTVIEW_WIDTH.

height

Height for this ListView. Default: DEFAULT_LISTVIEW_HEIGHT.

items

Initial list of items for this ListView. Default: empty list.

font

Font to be used for this ListView. Default: default Font constructor.

orientation

Orientation for this ListView. Default: Orientation.VERTICAL.

formatFunction

The formatFunction that is used to represent the items. Default: null.
