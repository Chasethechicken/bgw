TextField

open class TextField(height: Number, width: Number, posX: Number, posY: Number, label: String, font: Font, prompt: String) : LabeledUIComponent
Content copied to clipboard

A TextField is a single line input field. Whenever user input occurs the label field gets updated.

Parameters

height

height for this TextField. Default: TextField.DEFAULT_TEXT_FIELD_HEIGHT.

width

width for this TextField. Default: TextField.DEFAULT_TEXT_FIELD_WIDTH.

posX

horizontal coordinate for this TextField. Default: 0.

posY

vertical coordinate for this TextField. Default: 0.

label

initial label for this TextField. Default: empty String.

prompt

Prompt for this TextField. This gets displayed as a prompt to the user whenever the label is an empty string. Default: empty string.
