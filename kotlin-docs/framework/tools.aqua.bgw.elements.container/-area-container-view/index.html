AreaContainerView

open class AreaContainerView<T : GameElementView>(height: Number, width: Number, posX: Number, posY: Number, visual: Visual) : GameElementContainerView<T>
Content copied to clipboard

An AreaContainerView may be used to visualize a zone containing GameElementViews.

Visualization: The Visual is used to visualize a background. The positioning of the contained GameElementViews is used to place them relative to the top left corner of this AreaContainerView. Elements that are out of bounds for this AreaContainerView will still get rendered.

Parameters

height

height for this AreaContainerView. Default: 0.

width

width for this AreaContainerView. Default: 0.

posX

horizontal coordinate for this AreaContainerView. Default: 0.

posY

vertical coordinate for this AreaContainerView. Default: 0.

visual

visual for this AreaContainerView. Default: Visual.EMPTY.
