ComboBox

open class ComboBox<T>(height: Number, width: Number, posX: Number, posY: Number, prompt: String, font: Font, items: List<T>, formatFunction: (T) -> String?) : UIElementView
Content copied to clipboard

A standard ComboBox that may be populated with items of specified type parameter. The formatFunction is used to gain a String representation of each item. If no formatFunction is specified the toString function gets used instead.

Whenever the user selects an item, the selectedItemProperty gets updated.

Parameters

height

height for this ComboBox. Default: ComboBox.DEFAULT_COMBOBOX_HEIGHT.

width

width for this ComboBox. Default: ComboBox.DEFAULT_COMBOBOX_WIDTH.

posX

horizontal coordinate for this ComboBox. Default: 0.

posY

vertical coordinate for this ComboBox. Default: 0.

prompt

prompt for this ComboBox. This gets displayed as a prompt to the user whenever the selectedItemProperty value is null.Default: empty string.

font

font to be used for the prompt. Default: default Font constructor.

items

the initial selection of items. Default: empty list.

formatFunction

the formatFunction that is used to represent the items. Default: null.
