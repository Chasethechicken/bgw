class DiceAnimation<T : DiceView>(dice: T, toSide: Int, duration: Int, speed: Int) : ElementAnimation<T>
Content copied to clipboard

A dice roll Animation. Shuffles through dice visuals for given duration and shows toSide in the end. Use the speed parameter to define how many steps the animation should have. For example: An animation with duration = 1s and speed = 50 will change the visual 50 times within the duration of one second.

Parameters

dice

DiceView to animate.

toSide

resulting side after roll.

duration

duration in milliseconds. Default: 1 second.

speed

count of steps. Default: 50 steps.
